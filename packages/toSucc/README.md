## `@plexis/toSucc`

Converts a string to its equivalent successor.

**Installation**
`npm i @plexis/to-succ`

**Usage**

```javascript
import toSucc from '@plexis/to-succ';

toSucc('a'); // returns 'b'
toSucc('A'); // returns 'B'
toSucc('ABCD'); // returns 'BCDE'
```

**Aliases**

```javascript
import {toSucc} from 'plexis';
import {toSuccessor} from 'plexis';
```
